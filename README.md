# CloudMusicXposed #

网易云音乐Android客户端插件 - 单版本支持

- 自动签到
- 查看以往"每日推荐"歌单
- 阻止程序闪屏广告, 评论区广告(感谢 [zjns/PureNeteaseCloudMusic-Xposed](https://github.com/zjns/PureNeteaseCloudMusic-Xposed) )
- 去除因版权问题无法分享歌词图片的限制, VIP图片下载限制
- 阻止升级

**Download:**  [请自行编译](http://127.0.0.1/)
